## Введение
В компьютерах и других цифровых устройствах для представления и обработки информации используется дискретное (цифровое) представление данных. Наиболее универсальной формой представления данных является **двоичное кодирование**, состоящее из двух цифр: 0 и 1. Это обусловлено способностью устройств различать два состояния (например, замкнутое/разомкнутое, намагничено/ненамагничено и т.д.).

## Типы систем счисления
Системы счисления делятся на **позиционные** и **непозиционные**:

- **Непозиционные системы**: Значение цифры не зависит от её положения в числе. Пример: Римская система счисления (I = 1, V = 5, X = 10).
  
- **Позиционные системы**: Значение цифры зависит от её позиции в числе. Пример: десятичная система (1983₁₀ = 1×10³ + 9×10² + 8×10¹ + 3×10⁰).

## Позиционные системы счисления
### Основные понятия:
1. **Базис (или основание)**: Число, которое является основанием системы счисления. Например, в десятичной системе основание равно 10, в двоичной — 2.
2. **Разряд**: Позиция цифры в числе, от которой зависит её значение.
3. **Преобразование числа в позиционной системе**: Каждое число в системе записывается как сумма цифр, умноженных на соответствующие степени основания.

### Перевод целых чисел между системами счисления:
Чтобы перевести число из одной системы счисления в другую, нужно:
- Делить число на основание новой системы.
- Записывать остатки от деления, начиная с последнего.

#### Пример перевода:
**Перевод числа 143₁₀ в другие системы счисления**:
- В **двоичную**: 143₁₀ = 10001111₂
- В **восьмеричную**: 143₁₀ = 217₈
- В **шестнадцатеричную**: 143₁₀ = 8F₁₆

### Перевод двоичных чисел в другие системы:
- В **восьмеричную**: делим число на триады (группы по три бита).
- В **шестнадцатеричную**: делим число на тэтрады (группы по четыре бита).

#### Пример:
**Перевод числа 1101001111₂ в восьмеричную и шестнадцатеричную**:
- В восьмеричную: 1101001111₂ = 1517₈
- В шестнадцатеричную: 1101001111₂ = 34F,C₁₆

## Перевод десятичных дробных чисел
Для перевода десятичной дроби в другую систему счисления:
- Умножаем дробную часть на основание системы счисления.
- Результат округляем до целой части.

#### Пример:
**Перевод числа 0,5625₁₀ в двоичную систему**:
- 0,5625₁₀ = 0,1001₂

### Перевод дробных чисел из других систем в десятичную:
Для перевода дробного числа из другой системы счисления в десятичную:
- Применяем аналогичный метод, но учитываем степень основания для каждой цифры после запятой.

## Арифметические операции в системах счисления
### Сложение и вычитание:
Арифметические операции выполняются аналогично действиям в десятичной системе, но с учётом специфики каждой системы:

- В **двоичной** системе: 1 + 1 = 10₂, 100₂ - 1₂ = 11₂.
- В **восьмеричной** и **шестнадцатеричной** системах: операции аналогичны, но с учётом ограничений чисел в данных системах.

### Примеры:
- **Двоичные числа**: 100110011 + 1110110
- **Восьмеричные числа**: 127₈ + 43₈
- **Шестнадцатеричные числа**: 1B3C₁₆ + 567D₁₆

## Кодирование информации
### Условия Фано:
1. **Префикcный код**: Кодовое слово не может быть началом другого кодового слова.
2. **Постфиксный код**: Кодовое слово не может быть окончанием другого кодового слова.
