## **1. Тип `array` (массивы)**

- Хранят фиксированное количество элементов одного типа.
    
- **Пример:** 
    
    ```go
    var numbers [3]int = [3]int{1, 2, 3}
    fmt.Println("Массив:", numbers)
    ```
## **2. Тип `slice` (срезы)**

- Представляют динамический массив.
    
- **Пример:**
    
    ```go
    numbers := []int{1, 2, 3}
    numbers = append(numbers, 4) // Добавляем элемент
    fmt.Println("Срез:", numbers)
    ```

## **3. Карты (`map`)**

- Хранят данные в виде пар "ключ-значение".
    
- **Пример:** 
    
    ```go
    person := map[string]int{"Анна": 25, "Иван": 30}
    fmt.Println("Возраст Анны:", person["Анна"])
    ```
## **4. Тип `struct` (структуры)**

- Позволяют объединять различные типы данных в одну сущность.
    
- **Пример:** 
    
    ```go
    type Person struct {
        Name string
        Age  int
    }
    
    var p = Person{Name: "Анна", Age: 25}
    fmt.Println("Человек:", p)
    ```
## **5. Тип `interface`**

- Представляет набор методов, которые могут быть реализованы разными типами.
    
- **Пример:** 
    
    ```go
    var anyValue interface{}
    anyValue = 42
    fmt.Println(anyValue)
    anyValue = "Привет"
    fmt.Println(anyValue)
    ```
